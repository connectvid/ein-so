<Section>
<Capsule>**Form 5472 (Information Return of a 25% Foreign-Owned US Corporation) is required for ALL foreign-owned single-member LLCs.** File with a pro-forma Form 1120. The penalty for non-filing is $25,000 per form, per year. This is NOT optional — every non-resident who owns a US LLC must file Form 5472 annually.</Capsule>
</Section>

<Section surface>
<Label>What Is 5472</Label>

## What Is <Accent>Form 5472?</Accent>

| Detail | Information |
|--------|------------|
| Full Name | Information Return of a 25% Foreign-Owned US Corporation |
| Required for | Foreign-owned US LLCs and corporations |
| Filed with | Pro-forma Form 1120 |
| Due date | April 15 (extensions available) |
| Penalty | $25,000 per form per year |
| IRS form number | 5472 |

### Who Must File?
- Single-member LLCs owned by non-US persons
- US corporations with 25%+ foreign ownership
- Any US entity with reportable transactions with foreign related parties

### Reportable Transactions
Any monetary transaction between the US LLC and its foreign owner:
- Capital contributions
- Loans
- Payments for services
- Rent payments
- Any transfer of money or property

See [Form 5472 instructions](/form-5472-instructions/) for line-by-line guidance.
</Section>

<Section>
<Label>Filing</Label>

## How to File <Accent>Form 5472</Accent>

1. Complete Form 5472 (6 parts)
2. Attach to a pro-forma Form 1120 (even though LLCs normally don't file 1120)
3. File by April 15 of the following year
4. Request 6-month extension using Form 7004 if needed

### Filing Methods
- **Paper filing** — Mail to IRS
- **CPA/accountant** — Most non-residents use a US tax professional ($200-500)

**Important:** You need an [EIN](/apply/) before you can file Form 5472.
</Section>

<Section wide>
<Label>Penalties</Label>

## $25,000 Penalty for <Accent>Non-Filing</Accent>

The IRS imposes a $25,000 penalty for each Form 5472 not filed or filed incorrectly. This penalty:
- Applies per form, per year
- Can accumulate over multiple years
- Is assessed automatically
- Can sometimes be abated for reasonable cause

See [Form 5472 penalties](/form-5472-penalties/) for details on penalty abatement.
