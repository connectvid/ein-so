<Capsule>**Apply for an ITIN by completing Form W-7, gathering identity documents, and submitting with a federal tax return to the IRS.** Mail to: IRS ITIN Operation, PO Box 149342, Austin, TX 78714-9342. Processing takes 7-11 weeks. Use a Certifying Acceptance Agent (CAA) to keep your original passport.</Capsule>
</Section>

<Section surface>
<Label>Steps</Label>
## ITIN Application <Accent>Steps</Accent>

1. **Determine eligibility** — You need a US tax ID but cannot get an SSN
2. **Complete [Form W-7](/w7-form-guide/)** — Select correct reason code
3. **Prepare tax return** — Form 1040 or 1040-NR
4. **Gather ID documents** — Passport or 2 alternative IDs
5. **Submit** — Mail, CAA, or IRS office
6. **Wait 7-11 weeks** — IRS mails ITIN to your address

### Where to Mail
**IRS ITIN Operation**
**PO Box 149342**
**Austin, TX 78714-9342**
</Section>

## Common <Accent>Rejection Reasons</Accent>

| Reason | Fix |
|--------|-----|
| Missing tax return | Include 1040/1040-NR |
| Wrong reason code | Match your situation |
| Photocopies instead of originals | Send originals or use CAA |
| Missing signature | Sign Form W-7 |
| Name mismatch | Use exact passport name |
