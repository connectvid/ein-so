<Capsule>**Every LLC must file a Beneficial Ownership Information (BOI) report with FinCEN unless it qualifies for one of 23 exemptions.** Most LLCs do NOT qualify for exemptions (they have fewer than 20 employees and under $5M revenue). File free at fincen.gov/boi. Report all beneficial owners with 25%+ ownership or substantial control.</Capsule>
</Section>

<Section surface>
<Label>LLC Filing</Label>

## BOI Filing for <Accent>Your LLC</Accent>

### Step-by-Step
1. Go to fincen.gov/boi
2. Select 'File BOIR'
3. Enter company information (LLC name, EIN, address, state of formation)
4. Add each beneficial owner (name, DOB, address, ID document + photo)
5. Add company applicant (if formed after Jan 1, 2024)
6. Review and submit
7. Save your confirmation receipt

### What You Need
| Item | Details |
|------|---------|
| LLC name & EIN | As registered with state |
| State of formation | Where LLC was filed |
| Each owner's passport/license | Photo + number |
| Each owner's DOB & address | Residential, not business |
</Section>

<Section>
<Label>For Foreign LLCs</Label>

## Foreign-Owned <Accent>LLCs</Accent>

Foreign-owned LLCs have the same BOI requirements. Non-US beneficial owners report:
- Foreign passport as ID document
- Foreign residential address
- All other standard fields

BOI filing is separate from [Form 5472](/form-5472-guide/) (also required for foreign-owned LLCs).

See [EIN for non-residents](/ein-for-non-residents/) and [EIN for LLC](/ein-for-llc/).
</Section>

## Failure to File

- **$500/day** civil penalty
- Up to **$10,000** fine
- Up to **2 years** imprisonment (willful violations)
