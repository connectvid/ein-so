## How to Apply by <Accent>Fax</Accent>

Faxing Form SS-4 is the most reliable EIN application method for non-residents. No SSN required. Available from any country. Faster than mail (4-7 days vs 4-6 weeks). The IRS fax line operates 24/7 but processes forms during business hours.
</Section>

<Section>
<Label>Steps</Label>
## Step-by-Step <Accent>Fax Process</Accent>

1. **Download Form SS-4** from irs.gov — [SS-4 guide](/ss4-form-guide/)
2. **Complete all 21 fields** — Use passport number on Line 7b if no SSN
3. **Write a cover page** — Include return fax number and entity name
4. **Fax to [855-215-1627](/irs-fax-number-for-ein/)** — Confirm transmission
5. **Wait 4-7 business days** — IRS faxes your EIN back
6. **Save your EIN** — Use for [bank accounts](/ein-for-bank-account/), taxes, etc.

### Common Mistakes

| Mistake | Prevention |
|---------|------------|
| Missing Line 7b | Enter passport number |
| Wrong entity type | Match your LLC/corp docs |
| No return fax number | Include on cover page |
| Duplicate submission | Fax once only |
